# ⚽ REMATCH — 그날의 경기, 나의 전술로 다시

> 이미 끝난 실제 월드컵 경기의 감독석에 다시 앉아, 라인업과 전술을 바꾸고
> **"내가 감독이었다면"의 결과를 데이터 시뮬레이션으로 검증**하는 인터랙티브 웹서비스.

## 🎯 한 줄 소개

축구 팬의 가장 보편적인 감정은 **"만약에(What if)"**입니다. REMATCH는 술자리 논쟁으로만 소비되던
그 가정을, 실제 경기를 앵커로 삼는 **검증 가능한 경험**으로 바꿉니다.
경기를 고르고 → 선수를 배치하고 → 전술을 지시하고 → 경기 중 교체로 개입하고 →
**실제 역사와 나의 결과를 나란히 비교**하는 7~10분의 완결된 감독 체험 루프.

## ✨ 핵심 특징

- **8개 경기** — 2026 한국-남아공(0:1), 2022 한국-가나, 2018 한국-독일, 2002 한국-이탈리아, 2014 한국-알제리, 2014 브라질-독일(미네이랑), 2022 아르헨티나-프랑스 결승, 2010 한국-우루과이
- **드래그앤드롭 전술 보드** — 포인터 이벤트 단일 구현으로 데스크톱·모바일 동일 동작, 포지션 적합도 색상 피드백, 5개 포메이션 프리셋
- **분 단위 확률 시뮬레이션 엔진** — 배치·전술·체력이 곧바로 수식 입력값이 되는 순수 함수 모듈. xG 기반 슛 판정, 체력 모델, 전술 상성 매트릭스
- **실시간 감독 개입** — 교체 카드 3장(유한 자원), 하프타임 팀토크, 경기 중 전술 변경, 모멘텀 게이지
- **AI 코치** — 외부 API 없는 룰베이스 추천. 상대 약점을 근거 문장과 함께 제시
- **IF 리포트** — 실제 vs 나의 경기 비교 슬라이더, 감독 평점(S~F), 결과 카드 PNG 저장(Canvas)
- **감독 커리어 아카이브** — localStorage 기반. 서버·회원가입·결제 없이 브라우저에 기록 보존

## 🗺️ 화면 구성 (기획서 P1–P6 대응)

| ID | 화면 | 설명 |
|----|------|------|
| **P1** | 홈 | 6경기 라이브러리, 도발형 카피, 난이도·최고 평점 |
| **P2** | 프리매치 브리핑 | 실제 기록 타임라인, 상대 위협/약점 분석, 키플레이어 레이더 |
| **P3** | 작전실 | 드래그앤드롭 전술 보드, 5개 전술 다이얼, 예상 전력 게이지, AI 코치 |
| **P4** | 매치데이 | 라이브 피치+체력, 실시간 중계, 교체·하프타임·전술 개입, 모멘텀 |
| **P5** | IF 리포트 | 실제 vs 나의 경기 비교 슬라이더, 감독 평점, 결과 카드 저장 |
| **P6** | 아카이브 | 감독 커리어 요약, 도전 기록 리스트, 재도전 |

## 🎮 핵심 인터랙션 (기획서 INT-01–10 대응)

`INT-01` 선수 드래그앤드롭/탭 배치 · `INT-02` 포메이션 프리셋 재매핑 · `INT-03` 전술 슬라이더+트레이드오프 ·
`INT-04` 선수 카드 레이더 비교 · `INT-05` AI 코치 추천 · `INT-06` 시뮬레이션 배속 제어 ·
`INT-07` 인게임 교체(카드 3장) · `INT-08` 하프타임 팀토크 · `INT-09` 결과 카드 PNG 저장 ·
`INT-10` 실제 vs 나의 경기 비교 슬라이더

## 🛠️ 기술 스택

| 영역 | 선택 |
|------|------|
| 프레임워크 | React 18 + TypeScript + Vite 5 |
| 상태 관리 | Zustand |
| 애니메이션 | Framer Motion |
| 아이콘 | lucide-react (ISC License) |
| 폰트 | Pretendard (SIL OFL) |
| 데이터 | 정적 TS 모듈 + localStorage (**외부 API 0개**) |
| 품질 | Vitest 단위 테스트 + 몬테카를로 밸런스 스크립트 |

**아키텍처 핵심**: 시뮬레이션 엔진(`src/engine`)을 UI와 완전 분리된 순수 함수 모듈로 구현.
`(라인업, 전술, 시드) → 이벤트 로그`. 덕분에 엔진을 headless로 수백 회 돌리는 밸런스 검증이 가능하고,
심사자는 키 없이 브라우저에서 바로 플레이할 수 있습니다.

## 🚀 실행 방법

```bash
# 요구: Node.js 18+
npm install        # 의존성 설치
npm run dev        # 개발 서버 (http://localhost:5173)
npm run build      # 프로덕션 빌드 → dist/
npm run preview    # 빌드 결과 로컬 미리보기

npm test           # 엔진 단위 테스트 (Vitest)
npm run balance    # 몬테카를로 밸런스 검증 (경기별 400회 시뮬레이션)
```

## 📊 데이터 & 저작권 안내

- 선수·팀·경기 정보는 **공개된 사실 정보를 참조**로만 사용하며, 선수 **능력치는 REMATCH 자체 산정값**(0~99)으로 공식 기록이 아닙니다.
- 선수 사진·클럽 로고 등 저작권 이미지는 사용하지 않으며, 등번호·국기 이모지·자체 SVG로 표현합니다.
- 특정 선수·팀·국가를 비하하지 않으며, 패배 서사도 중립적 어휘로 작성합니다.
- 외부 유료 API·회원가입·결제 없이 심사 가능합니다.
