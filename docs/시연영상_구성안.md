## 촬영 팁

- **추천 도전 경기**: 브라질 vs 독일(미네이랑) — "7실점을 막아라"는 극적인 서사가 심사 임팩트가 큼

- 골 장면은 배속 ×1로 두어 골 플래시 연출을 온전히 담기
- 모바일 반응형도 짧게 보여주면 크로스 디바이스 완성도 어필 가능
